检查是否有安装samba服务：

$ rpm -qa | grep samba

如果没安装则yum安装

$ yum -y install samba samba-client samba-common

默认的安装目录是/etc/samba，配置文件为smb.conf,先备份下smb文件
$ cd /etc/samba
$ cp smb.conf smb.conf.bak
